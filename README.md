# Laboratorio 14 - Desarrollo de la Base de un E-commerce

## Descripción

En este laboratorio se desarrolló la estructura base de una aplicación de comercio electrónico utilizando React y TypeScript. Se configuró un proyecto moderno con Vite y se integró la biblioteca de componentes shadcn/ui para facilitar la construcción de interfaces reutilizables.

Asimismo, se implementó un sistema de rutas para la navegación entre páginas, un layout reutilizable para mantener una estructura consistente en toda la aplicación y una Homepage inicial que servirá como base para el desarrollo futuro del e-commerce y la incorporación de funcionalidades relacionadas con productos y transacciones.
